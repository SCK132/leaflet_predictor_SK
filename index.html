WASA 気球実験用 着地点予測ツール (CUSF/Tawhiri ベース Fork) (元プロジェクト情報)

ラテックス観測気球の飛行経路と着地点を予測するツールです。

本ツールは CUSF 向けに Jon Sowman / Adam Greig / Daniel Richman らが構築し、Rich Wareham 他の貢献を含むオリジナルコードを基に WASA 気球実験向けに日本語化・機能拡張 (JST 対応 / 愛媛モード 等) した派生版です。旧版由来部分には Rob Anderson / Fergus Noble / Ed Moore らのコードが含まれます。

Leaflet への移行と Tawhiri API 利用は Mark Jessop により SondeHub プロジェクトとして実施。Tawhiri API の Sondehub インスタンス は非商用用途で利用可能です。

この予測ツールはNOAA GFSモデルのデータを使用しています。

本ソフトウェアによって生成されるデータの正確性、精度、信頼性については一切保証されず、利用者は自己責任で使用するものとします。詳細については、libera.chatの #highaltitude を参照してください。

この予測ツールは非営利目的での利用のみを意図しています。商用環境で使用する場合は、こちらのページをお読みください。

破裂計算機

Burst Calculator Web Page

Usage

この計算機は、高高度気球に充填するガスの量を算出するために設計されています。これにより、目標とする破裂高度や上昇速度を達成できます。必要なのは、ペイロード質量（パラシュートを含む）と気球の質量です。

使い方は非常に簡単です。最初のボックスにペイロード質量を入力し、2番目のボックスから気球の質量を選択します。 そして、目標とする破裂高度（メートル）か、上昇速度（メートル/秒）のどちらかを該当するボックスに入力します。 Enterキーを押すか、入力ボックスの外をクリックすると計算が実行され、結果が下のボックスに表示されます。

「定数」ボックスを使用して、計算機で使用される定数を調整できます。 ガスの種類の選択（またはカスタムガス密度の入力）、モデリングパラメータの調整、または気球のカスタム破裂直径や抗力係数の入力が含まれます。 カスタム値を使用するには、破裂直径または抗力係数のボックスの横にあるチェックボックスをオンにしてください。それ以外の場合は、選択した気球に適切な値が使用されます。

概要

CUSFのために2010年3月にAdam Greigによって作成されました。数学的処理はSteve Randallによる `burst1a` スプレッドシートから派生しています。気球情報はKaymont Totexのサウンディング気球データに基づいています。

本プログラムに含まれる、または生成されるデータの正確性については一切保証されません。自己責任でご使用ください。
